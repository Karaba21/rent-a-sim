## Project Overview

This is a single-page website for a reservations-based business.

The main goal of the site is to present the business clearly, build trust, and drive visitors to book a reservation through a Calendly integration.

Keep the website simple, professional, fast, and easy to maintain.

## Tech Stack

- Next.js

- TypeScript

- Tailwind CSS

- Single-page layout

- Future integration with Calendly
